JavaScript初识

JavaScript是Web页面中的一种脚本编程语言，也是一种通用的、跨平台的、基于对象和事件驱动并具有安全性的脚本语言。它不需要进行编译，而是直接嵌入在HTML页面中，把静态页面转变成支持用户交互并响应相应事件的动态页面。

在HTML中的使用

在页面中直接嵌入JavaScript

​ 在HTML文档中可以使用 <script>... </script>标记将JavaScript脚本嵌入到其中。
​script属性：

属性 说明
language 不常用
type 用于指定使用的脚本类型
src 指定外部脚本文件路径
defer 不常用

type属性的语法格式：

<script type = "text/javascript"></script>

链接外部JavaScript文件

​ 语法格式：多了个src属性

<script type = "text/javascript" src = "js文件的rul路径"></script>

注意：

  1. 外部JavaScript文件中只能包含JavaScript代码，不能包含HTML和<script>标签。
  2. 在<script>标签内部不要存在其他的JavaScript代码。
  3. 在链接外部JavaScript文件时</script>结束标签不能省略。

作为标签的属性值使用

在HTML文档中可以在<a>标签、<input>标签 中使用JavaScript脚本作为它们的属性值。
通过“JavaScript”调用

<a href="javascript:alert('你好JavaScript')">测试</a>

​ 与实践结合调用

<input type="button" value="测试" onclick="alert('你好JavaScript ')" />

分支语句 (switch)

switch 它根据一个表达式的值，选择不同的分支执行。语法格式：

switch (表达式){
    case 常量表达式1 :
        语句1;
        ···;
        break;
    case 常量表达式2 :
        语句2;
        ···;
        break;
        ···
    case 常量表达式n :
        语句n;
        ···;
        break;
    default:
        语句n+1;
        break;
}

default语句可以省略。在表达式的值不能与任何一个case语句中的值相匹配的情况下，JavaScript会直接结束switch语句，不进行任何操作。
case后面常量表达式的数据类型必须与表达式的数据类型相同，否则匹配会全部失败，而去执行default语句中的内容。
在switch语句中漏写break语句。则会影响其他语句

循环语句

循环语句 (while)

while 判断，它是利用一个条件来控制是否要继续重复执行这个语句。语法格式：

while(表达式)(
   语句（循环体）
)

在使用while语句时，一定要保证循环可以正常结束，即必须保证条件表达式的值存在为false的情况，否则将形成死循环。

循环语句 (do…while)

do…while 后判断，它也是利用一个条件来控制是否要继续重复执行这个语句。语法格式：

do{
   语句
}while(表达式);

循环语句 (for)

for 计次循环，一般用于循环次数已知的情况。语法格式：

if(初始化表达式;条件表达式;迭代表达式){
    语句（循环体）
}

在使用for语句时，也一定要保证循环可以正常结束，也就是必须保证循环条件的结果存在为false的情况，否则循环体将无休止地执行下去，从而形成死循环。

跳转语句

continue语句

用于跳过本次循环，并开始下一次循环。
当使用continue语句跳过本次循环后，如果循环条件的结果为false,则退出
循环，否则继续下一次循环。
continue语句只能应用在while、for、do…while语句中.

break语句

在循环语句中，break语句用于跳出循环。

continue语句只能应用在while、for、do…while或switch语句中。在嵌套的循环语句中，break语句只能跳出当前这一层的循环语句，而不是跳出所有的循环语句。

数值处理函数

字符转数字函数 ( parseInt )

parseInt 将首位为数字的字符串转换成数字，字符串开头不是数字的字符，则返回NaN，前提是默认十进制。语法格式：（默认十进制）

parseInt(String,[进制数]) 
如: parseInt("e2b",16)
结果为:3627

字符转浮点函数 (parseFloat)

parseFloat 将首位为数字的字符串转化成浮点型数字，字符串不是以数字开头，则返回NaN。语法格式：

parseFloat(String)
如: parseFloat("5.411")

检验函数 (isNaN)

isNaN 检验某个值是否为NaN。语法格式：

isNaN(num)

返回值：
结果为NaN 函数返回值为true
否则，函数返回值为false

检验限度函数 (isFinite)

isFinite 检验其参数是否有限。语法格式：

isFinite(num)

num：需要验证的数字

返回值：
有限数字，则返回true
NaN或无穷大，则返回false

escape 将一些特殊字符(不包括字母和数字字符以及、@、一、_、+、.和/)*进行编码，它可以将这些特殊字符转换为“%XX**” 格式的数字。语法格式：

unescape 对应用escape()函数编码后的字符串进行解码。它可以将字符串中“%XX”格式的数字转换为字符。语法格式：

unescape(String)
//如: unescape("You%20%26%20Me") →You & Me

event对象

用来描述JavaScript的事件。事件一旦发生，便会生成event对象

event 对象引用

window对象属性的访问：

window.event

全局引用对待event对象：

event.propertyName

事件处理时没有特殊情况的话，都可以访问event对象

W3C的引用

将 event对象 作为一个参数传递到事件处理函数中。

将事件处理函数与对象绑定的前提要把 event对象 作为参数传递 。例子：

onKeyUp = "example(event)"

在W3C模型中全局引用 event对象 的方式

多参数，例子：

onKeyUp = "example(this.event)"

将事件处理函数的引用 赋值给文档中需要的对象，例子：

document.forms[0].someButton.onKeyUp = example;
document.getElementByld("myButton").addEventListener("KeyUp" , example , false);

event 对象属性

AltLeft 属性。获取左 Alt键 的状态，返回布尔值：true 关闭，false 不关闭

[window.]event.altLeft

CtrlLeft 属性。获取左 Ctrl键 的状态，返回布尔值：true 关闭，false 不关闭

[window.]event.CtrlLeft

ShiftLeft 属性。获取左 Shift键 的状态，返回布尔值：true 关闭，false 不关闭

[window.]event.ShiftLeft

示范例子：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script type="text/javascript">
        //定义函数
        function exa() {
            //按下按钮的条件
            if (window.event.ctrlKey && window.event.altKey && window.event.shiftKey){
                alert("快乐学习，JS！");
                //链接 URL 路径
                window.location.href = 'http://www.baidu.com';
                }
            }
            if (window.event.ctrlKey){
                alert(window.event.ctrlKey+" , 按下了Ctrl键")
            }
        //应用 onkeydown 事件输出函数中的内容
        document.onkeydown = exa;
    </script>
</head>
<body>
    <p> 按下 Ctrl键 + Alt键 + Shift键 百度搜！</p>
</body>
</html>

button 属性。获取事件发生时用户所按的 鼠标键 

[window.]event.button

属性值：

button 值 说明
0 没有 按键
1 按下 左键（主键）
2 按下 右键
3 同时按下 左键 和 右键
4 按下 中键
5 同时按下 左键 和 中键
6 同时按下 左键 和 中键
7 同时按下 左键 、中键 、 右键

个人测试的 button值

button 值 说明
0 按下 左键（主键）
1 按下 中键
2 按下 右键
3 按下 侧键5号键
4 按下 侧键4号键

注意 ：

如果同时按下两个按键执行的特殊动作，会干扰到目标行为。
